# Project Cemetery / Кладбище Проектов

**Project Cemetery** — это цифровое пространство для похорон заброшенных, провальных или просто нереализованных проектов. Здесь вы можете официально «похоронить» свою идею, написать эпитафию и получить поддержку сообщества в виде виртуальных свечей.

##  Основная концепция

Каждый добавленный проект получает своё виртуальное **надгробие**, содержащее:
-  **Название и описание** — суть идеи
-  **Дата «захоронения»** — когда проект был официально закрыт
-  **Причина провала** — честный разбор того, что пошло не так
-  **Эпитафия** — короткая фраза на память
-  **Технологии** — стек, на котором проект создавался
-  **Стадия провала** — на каком этапе всё остановилось (`Идея`, `Прототип`, `Разработка`, `Тестирование`, `Запуск`, `Масштабирование`)
-  **Press F** — возможность для других посетителей «поставить свечу» в знак уважения и поддержки

##  Функционал

-  **Авторизация и аккаунты** — регистрация, вход/выход, сохранение сессии
-  **Стена памяти** — возможность оставлять слова поддержки и комментарии под каждым надгробием
-  **Архив публикаций** — лента всех захороненных проектов
-  **Поиск и фильтрация** — по названию, стадии провала или технологиям
-  **JWT-токены** — безопасное управление сессиями и возможность их обновления
